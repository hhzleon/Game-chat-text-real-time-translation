# Game-chat-text-real-time-translation
各位大佬大家好，本翻译器有诸多不完善的地方。没办法，这是写着玩的，多多包涵。



这个翻译器的原理就是ocr+翻译接口，东西很简单。给弄了个界面，大佬们拉下来然后改改就能用了。也欢迎大佬们基于这个模板完善功能，让语言不再是游戏的障碍。

![](./img/1.png)

## 配置教程

1. 拉下来文档
2. 安装ocr环境
3. 申请百度接口
